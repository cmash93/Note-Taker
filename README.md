## Installation 

Within your console:    
```
npm init
```    
```
npm install express
```    
```
npm install uuid
```    

## Usage
*If running site on local host*
Within your console:    
```
node server.js
```   
```
App listening at http//localhost:3001
```   
Visit the link where the app is listening, and you'll be able to begin writing notes!
